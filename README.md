## Hand-To-Hand server

* **URL**
  
  <_/votes_>

* **Method:**
  
  `GET` | `PATCH`
  
* **Required:**
  
  `id=[integer]`
  
* **Success Response:**
  
  * **Code:** 200
    **Content** `{ id : 12 }`
    
* **Error Response:**
  
  * **Code:** 401 UNAUTHORIZED
